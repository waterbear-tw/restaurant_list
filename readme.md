# Restaurant List

![web screenshot](./public/screenshot.png)

## 功能

- 查看餐廳清單、餐廳種類及其評分
- 查看餐廳的詳細資訊
- 透過關鍵字搜尋到相關餐廳名稱

## 使用方式

1. 安裝 Node.js 以及 npm
2. clone 份專案
3. 在專案資料夾中安裝 package.json 中的模組:

```
npm install
```

4. 執行網站，使用 nodemon 啟動網站伺服器

```
nodemon app.js
```

5. 開啟瀏覽器輸入 localhost:3000 即可執行網站

## 開發工具

- Node.js: 18.5.0
- npm: 9.5.0
- Express: 4.16.4
- Express-handlebars: 3.0.0
- Bootstrap: 5.1.3
